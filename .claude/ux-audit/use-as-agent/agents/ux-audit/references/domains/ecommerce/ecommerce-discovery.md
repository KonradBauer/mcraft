## E-commerce Discovery Guidance

### Key Assumptions to Form
- **Catalog size**: <50 SKUs (curated) / 50-500 (medium) / 500+ (large catalog)
- **Average order value**: <$30 (impulse) / $30-100 (considered) / $100+ (high-consideration)
- **Purchase model**: One-time / subscription / mixed
- **Primary traffic source**: Paid social / organic search / email / direct
- **Mobile percentage**: Assume 65%+ unless told otherwise
- **Return/exchange rate**: High → sizing/fit is a conversion barrier → need size guide
- **Repeat purchase rate**: High → loyalty/reorder UX matters; low → first-purchase conversion is everything

### Common Project Shapes
- **Shopify DTC brand**: Transaction-First archetype, 20-100 SKUs, social media traffic
- **Large retailer**: Catalog-Led archetype, 500+ SKUs, search-heavy traffic
- **Premium DTC**: Brand-Story archetype, curated catalog, story-driven
- **Subscription box**: Subscription-First archetype, quiz/customization flow
- **Marketplace seller**: Transaction-First with platform constraints

### Unique Constraints
- **Product photography**: The #1 conversion factor — poor images cannot be compensated by good UX
- **Shipping/returns**: Must be communicated early (hidden costs = #1 cart abandonment reason)
- **Payment options**: Express checkout (Apple Pay, Klarna) significantly impacts mobile conversion
- **Inventory status**: Real-time stock display creates urgency and prevents disappointment
- **Seasonal cycles**: Site needs to accommodate promotional periods without permanent urgency fatigue

### Default Assumptions
If no info provided: Shopify-based DTC brand, 20-100 SKUs, AOV $40-80, traffic split 50% paid social / 30% organic / 20% email, mobile-dominant (65%+ traffic), free shipping threshold at $75, return rate ~15%.
