## Event Discovery Guidance

### Key Assumptions to Form
- **Event format**: In-person / virtual / hybrid?
- **Pricing model**: Free / paid / tiered pricing?
- **Frequency**: One-time / recurring series / annual?
- **Deadline structure**: Registration close date? Early-bird deadline? Limited seats?
- **Target attendee**: Who and what's the primary draw — speakers, content, networking, entertainment?
- **Multiple audiences**: Attendees only, or also sponsors, vendors, press?
- **Expected scale**: <50 (intimate) / 50-500 (medium) / 500+ (large conference)?

### Common Project Shapes
- **Industry conference**: Urgency-Driven archetype, paid tiers, speaker-focused, 100+ attendees
- **Webinar / virtual event**: Urgency-Driven archetype, free/low-cost, single topic, speaker authority
- **Monthly meetup**: Community-Event archetype, free, recurring, community-building
- **Product launch**: Launch-Page archetype, waitlist/pre-order, countdown, anticipation
- **Annual conference with year-round content**: Hybrid archetype, full site + event hub

### Unique Constraints
- **Time-bound nature**: Page has an expiration date — plan for post-event state (redirect, archive, recording access)
- **Single-CTA discipline**: Resist adding navigation, content sections, or secondary goals. One page, one goal = 3.7× more signups
- **Countdown authenticity**: Fake urgency permanently destroys trust. Only use real deadlines
- **Calendar integration**: Post-registration, provide .ics file and calendar add buttons — reduces no-show rate
- **Share mechanics**: After registration, prompt "Invite a colleague" with pre-written message — peer invitations are the #1 acquisition channel for events
- **Mobile registration**: 50%+ of event traffic is mobile (from email/social) — form must be flawless on small screens

### Default Assumptions
If no info provided: Single in-person event, paid registration with 2-3 ticket tiers, early-bird and general pricing, 50-500 expected attendees, primary draw is speaker lineup, single audience type (attendees), mobile traffic 50%+.
